# other thoughts

虽然CSS有仔细学，但在编写页面时还是需要ai辅助。编写过程感觉很久很难想，但是编写完感觉还是很简陋。要想达到自己想要的效果，还得更加深入的学习以及配合JavaScript的使用。vs code上的ai十分便捷，能省去大量编写时间同时还能检查错误。

**别依赖 AI**：自己先试着写页面，哪怕写得丑，先把布局、样式的逻辑捋清楚，实在卡壳了再用 AI 找思路，而不是直接抄。

**CSS不能只停留在开始阶段**：要想做出美观的页面必须要往深处学习CSS ，也是一名前端人员应该掌握的技能。

**搭配 JS 一起练**：学点简单的 JS，比如用 JS 做个点击按钮换颜色、显示隐藏内容的效果，慢慢把 CSS 和 JS 结合起来用。
